# Pokretanje projekta-Punjenje baze

---

# 1.Preduslovi

Podatke preuzmite sa https://www.kaggle.com/datasets/artyomkruglov/gaming-profiles-2025-steam-playstation-xbox  

Potrebne su sve datoteke iz `playstation` foldera

Datoteke (.csv) smestite u okviru `data` foldera

U terminalu pokrenite(može u aktivnom virtuelnom okruženju)

```bash
pip install -r requirements.txt
```

---

# 2.Pokretanje skripte

---

Pokrenite u terminalu

```bash
python main.py
```
